API 연결 설정
Railway 백엔드 서버 연결 · 데이터 동기화
① 백엔드 서버 주소
Railway 배포 후 받은 URL을 입력하세요.
② 수동 동기화
📋 사용법: 서버 URL 저장 후 아래 버튼을 눌러 원하는 데이터를 즉시 가져옵니다. API 키가 쿠팡에서 활성화된 후(발급 후 최대 24시간) 사용 가능합니다.
📦 상품 동기화
쿠팡에 등록된 상품·옵션 목록을 불러옵니다.
📊 판매 동기화
최근 정산 내역(매출·수량)을 불러옵니다. 날짜 지정 가능.
📣 광고비 동기화
일별 광고비 소진 내역을 불러옵니다.
📜 동기화 로그
③ 자동 동기화 설정
⏱ 자동 동기화: 앱이 열려 있는 동안 지정한 주기마다 판매·광고 데이터를 자동으로 불러옵니다. 브라우저 탭이 닫히면 멈추며, 다시 열면 재시작됩니다.
자동 동기화 비활성화 상태
📌 전체 사용 가이드
1단계 — Railway에 백엔드 배포 → 서버 URL 입력 후 저장
2단계 — 쿠팡 Wing → 개발자센터 → API 키 발급 (활성화 최대 24시간 소요)
3단계 — 연결 테스트 ✅ 확인 후 수동 동기화로 초기 데이터 수집
4단계 — 자동 동기화 ON → 주기·항목 선택 → 이후 자동 운영
※ 판매 데이터는 쿠팡 정산 기준이라 당일 포함 1~2일 지연될 수 있습니다.
